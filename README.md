# Rajarshi Janak University Library Management System

A comprehensive web-based library management system for Rajarshi Janak University.

## Features

- **Book Management**: Add, edit, delete, and search books
- **Student Management**: Manage student records and generate student IDs
- **Book Checkout/Checkin**: Easy book borrowing and return system
- **Transaction Records**: Complete history of all library transactions
- **Admin Panel**: System management and course administration
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Free Search**: Advanced search functionality for books and students

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (ES6+)
- LocalStorage for data persistence
- Responsive design with mobile-first approach

## Installation

1. Clone or download this repository
2. Open `index.html` in a web browser
3. The system is ready to use!
